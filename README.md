🚀 **EZ Cyber Tool** is a lightweight cybersecurity toolkit written in **Bash**.  
It includes useful **network, OSINT and security utilities** inside a simple CLI menu.

---

# ✨ Features

- 🌐 IP Lookup
- 📡 Ping Test
- 🔍 Port Scanner
- 🌍 DNS Lookup
- 🛰 Whois Lookup
- 🔐 Password Generator
- 🧠 Password Strength Checker
- 🌐 Website Info Scanner
- 🔎 Subdomain Finder
- 💻 System Information
- 🌍 Public IP Finder

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/setvotka/ezcybertool.git

Enter the directory:

cd ezcybertool

Give execution permission:

chmod +x main.sh

Run the tool:

bash main.sh
🖥 Requirements

Bash

curl

nmap

whois

Install in Termux:

pkg install curl nmap whois
